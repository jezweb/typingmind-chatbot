Typing Indicator Test

Testing Typing Indicator Feature

This page tests the typing indicator functionality:

  • Typing indicator should appear when sending a message
  • Three animated dots should show while waiting for response
  • Indicator should disappear when response starts streaming
  • Works in both popup and inline modes

What to Look For

When you send a message, you should see:

  1. Three animated dots appear below your message
  2. The dots continue animating while the API processes
  3. The dots disappear and are replaced by the actual response
