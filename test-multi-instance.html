How This Works

The multi-instance system allows you to:

  • Use instance IDs instead of TypingMind agent IDs
  • Have multiple configurations for the same TypingMind agent
  • Clone agents in TypingMind without breaking your widgets
  • Set different themes, domains, and rate limits per instance

Update Your Embed Code

Change from:

TypingMindChat.init({
    agentId: 'character-xxx-xxx',  // TypingMind agent ID
    embedMode: 'inline'
});

To:

TypingMindChat.init({
    instanceId: 'seo-bot-test',    // Your instance ID
    embedMode: 'inline'
});
